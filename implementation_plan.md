# خطة تنفيذ مشروع NEXORA - نظام إدارة مكتبة الوسائط الذكية

تم إعداد هذا المستند كخطة عمل وتصميم للنظام المقترح لإدارة مكتبات الوسائط الضخمة في الاستراحات وصالات الألعاب مع إعادة تصميم البنية التقنية لتناسب نموذج (الخادم المركزي والعملاء) والتغلب على قيود Flutter.

## مراجعة وتأكيد من المستخدم المطلوب (User Review Required)

> [!IMPORTANT]
> يرجى مراجعة القرارات المعمارية التالية والموافقة عليها قبل البدء في مرحلة البرمجة الفعلية:
> 1. **استبعاد Flutter واستبداله بـ React (الويب):** هذا القرار يلغي الحاجة لتثبيت أي برمجيات على أجهزة العملاء (الزبائن) بالاستراحة. بمجرد فتح متصفح الويب على الرابط المحلي للسيرفر، تظهر الواجهة السينمائية الفاخرة للزبون كاملة وتعمل فوراً.
> 2. **استخدام لغة Go (Golang) للخلفية والسيرفر:** لغة Go تعطي أداءً فائقاً جداً مع التزامن ومعالجة الملفات الكبيرة (100+ TB)، مما يضمن عدم حدوث بطء في السيرفر أثناء قراءة البيانات وتسييلها للأجهزة.
> 3. **آلية تشغيل الفيديو (Direct Streaming via LAN):** سيتم بث الفيديو مباشرة من السيرفر للأجهزة الفرعية عبر بروتوكول HTTP بمشغل فيديو متطور يدعم تغيير مسارات الترجمة والصوت ديناميكياً.

---

## المكونات المقترحة للتعديل والإنشاء (Proposed Components)

### 1. خادم الخلفية الذكي (Server API & Indexer)
- **لغة البرمجة:** Go (Golang)
- **قاعدة البيانات:** PostgreSQL
- **محرك البحث:** Meilisearch (للبحث اللحظي مع دعم اللغتين وتصحيح الأخطاء الإملائية).
- **المكتبات المساعدة:** FFmpeg (لتوليد الصور المصغرة والتحقق من سلامة الفيديو)، و MediaInfo (لقراءة تفاصيل الجودة والأكواد الفنية).

### 2. واجهة المستخدم النهائية ولوحة التحكم (Frontend Client & Admin Panel)
- **التقنيات المستخدمة:** React + Tailwind CSS + Framer Motion (للحصول على تصميم زجاجي فاخر Neon Glassmorphism وتأثيرات حركية بالغة النعومة).
- **مشغل الفيديو:** مشغل ويب متطور (Plyr / Video.js) مخصص لدعم مسارات ترجمة خارجية وصوت متعددة.

## خطة التحقق والجاهزية (Verification Plan)

### الاختبارات البرمجية الذاتية (Automated Tests)
- كتابة اختبارات وحدة (Unit Tests) لـ **Regex Parser** للتحقق من دقته في استخراج تفاصيل الأسماء العربية والإنجليزية للمسلسلات والأفلام.
- اختبار سرعة الفهرسة للتأكد من قدرة السيرفر على فهرسة آلاف الملفات في ثوانٍ معدودة.

### التحقق اليدوي (Manual Verification)
- التحقق من تشغيل الفيديو وبثه عبر الشبكة المحلية (LAN) إلى جهاز فرعي آخر دون تقطيع.
- تجربة الواجهة والتأكد من مطابقتها للتصميم الفاخر المعروض في صور المشروع.
- اختبار البحث اللحظي عند إدخال كلمات باللغة العربية والإنجليزية وبأخطاء إملائية خفيفة.
